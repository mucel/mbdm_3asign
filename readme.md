# Evaluation of network structers in relation to policies. 
By Noah Zuijderwijk and Māra Učelniece

Amsterdam


In this assignment we analysed the transition to electric vehicles (EVs) using a networked agent-based model. The provided Mesa implementation combines a Stag- Hunt coordination game with positive feedback from charging infrastructure.
